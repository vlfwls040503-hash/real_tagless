### Q1.5 적용 범위 명시 (작성 필요)

본 시뮬레이션의 적용 범위:
- **흐름 유형**: 단방향 출구 흐름 (계단 → 게이트 → 출구). 양방향 흐름(입구+출구 동시)은 미반영
- **밀도 범위**: 0 ~ 3.0 P/m². 초고밀도(>3.0) 상황은 CFSM V2의 검증 범위 밖
- **보행자 유형**: 건강한 성인. 유모차, 휠체어, 대형 짐 보행자 미반영
- **공간 범위**: 성수역 서쪽 대합실 (50m × 25m). 에스컬레이터/엘리베이터 미포함
- **시간 범위**: 열차 1~2회 도착 (300초). 장시간 정상상태 미고려

### 모델 전환 근거: GCFM → CFSM V2

**전환 사유**:
- **진동 문제**: GCFM은 힘 기반(F=ma) 2차 모델로, 보행자 간 반발력이 진동을 유발함 (Kretz, 2015)
- **겹침 문제**: V6 검증에서 최대 22.4cm 겹침 발생 (Level C FAIL). max_repulsion_force 상한으로 인해 고밀도에서 충돌 방지 불가
- **dt 민감도 문제**: V5 검증에서 dt 변경 시 유량 24% 변동 (FAIL). 힘 적분 정밀도가 결과에 과도하게 영향

**CFSM V2 선택 근거** (Tordeux et al., 2016):
- **충돌 방지 보장**: 1차 속도 모델로 보행자 간 겹침이 구조적으로 발생하지 않음
- **dt 안정성**: 힘 적분이 없으므로 dt 변경에 대한 수치적 민감도가 낮음
- **JuPedSim 공식 게이트 예제 존재**: JuPedSim에서 CFSM V2 기반 게이트 시나리오를 공식 예제로 제공

**대기행렬 형성 확인**:
- CFSM V2 + 서비스 시간 모델로 대기행렬 정상 형성 확인 (피크 시 최대 7명 대기)

**time_gap=0.80 캘리브레이션**:
- Seyfried (2009) 병목 유량 실험 데이터에 맞추어 캘리브레이션
- Tordeux (2016) 원논문 기본값 1.06에서 0.80으로 조정하여 실측 유량 재현

| ID | 테스트 | 방법 | 통과 기준 | 근거 | 상태 |

| **V1** | **자유보행 속도** | 30m×5m 복도, 단일 보행자, 6개 희망속도 | 실측/희망 오차 < 5% | RiMEA Test 1 | ✅ PASS (오차 0.0%) |

### V4 게이트 선택 규칙 검증 — 상세 설계

**목적**: MNL 모델이 코드에 올바르게 구현되었는지 확인

**테스트 A: 거리 효과**
- 보행자 1명, 게이트 2개 (5m vs 10m), 대기열 모두 0
- 100회 반복, 근거리 게이트 선택 비율 기록
- 기대값: P(근거리) = exp(β_d × 5) / [exp(β_d × 5) + exp(β_d × 10)]
- 통과 기준: 관측 비율과 이론 비율의 χ² 검정 p > 0.05

**테스트 B: 대기열 효과**
- 보행자 1명, 게이트 2개 (동일 거리 5m), 대기열 0 vs 5명
- 기대값: MNL 이론 확률과 일치
- 통과 기준: χ² 검정 p > 0.05

**테스트 C: 3종 성격 민감도**
- adventurous/conserved/mild 각 100명, 동일 조건
- 기대값: adventurous가 먼 빈 게이트 선택 확률 > conserved
- 통과 기준: 성격별 선택 비율 순서가 이론 예측과 일치

### V5 수치 수렴성 — CFSM V2 전환으로 해결됨

**기존 GCFM 결과** (참고용):
- GCFM에서 dt=0.01 → 0.005 변경 시 유량 24.1% 변동으로 FAIL 판정
- 원인: GCFM은 2차 미분방정식(F=ma) 기반으로 dt 변경 시 힘 적분 정밀도가 결과에 과도하게 영향

**CFSM V2 전환 후**:
- CFSM V2는 1차 속도 모델(v = f(간격))로 힘 적분이 없음
- 구조적으로 dt 민감도가 낮아 PASS가 예상되나, 정량적 재측정 필요
- 재측정 후 결과 업데이트 예정

### V6 물리 일관성 — CFSM V2 전환으로 해결됨

**기존 GCFM 결과** (참고용):
- GCFM에서 겹침 153건 발생, 최대 22.4cm (Level C FAIL)
- 원인: GCFM은 힘 기반 모델로 충돌을 원천 방지하지 못함. max_repulsion_force 상한 도달 시 추가 반발 불가

**CFSM V2 전환 후 결과**:

| 항목 | 값 |
|------|-----|
| 에이전트 수 | 120명 (밀도 1.5 P/m²) |
| 겹침 발생 쌍 | **0건** |
| 최대 겹침 | **0.0 cm** |
| 판정 | **✅ PASS** |

CFSM V2는 1차 속도 모델로, 보행자 간 간격이 줄어들면 속도가 자동으로 0에 수렴하여 겹침이 구조적으로 발생하지 않는다. 이는 CFSM의 핵심 설계 특성이다.

## Phase 4: Validation (타당성 검증 — 현실과 맞는가?)

> NIST TN 1822 §3.2: "캘리브레이션에 사용하지 않은 독립 데이터로 비교"

### U1 반복 수렴성 — 상세 설계 (ERD 방법)

> Ronchi et al. (2013) NIST TN 1822 §4.1 권장

1. 기준 시나리오를 N = 1, 2, ..., 50회 반복
2. 각 N에서 누적 평균 대기시간 계산: $\bar{X}_N = \frac{1}{N}\sum_{i=1}^{N} X_i$
3. ERD 계산: $ERD_N = \frac{|\bar{X}_N - \bar{X}_{N-1}|}{\bar{X}_{N-1}} \times 100\%$
4. ERD < 5%가 **연속 3회** 유지되는 최소 N을 채택
5. 모든 시나리오를 해당 N회 이상 반복
