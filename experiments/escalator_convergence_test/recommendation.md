## 최종 권고: 전략 A (Funnel geometry) — **A_funnel_5m_15°**

### 선정 근거

1. **성능**: 진동 지표 40% 감소(208 → 126 deg/s) + **처리량 증가** (34 → 37 exited)
2. **현실성**: 실제 지하철 에스컬레이터 앞에는 안전 가이드·난간이 존재. funnel은 이를 충실히 재현
3. **구현 비용 최소**:
   - `seongsu_west.py` STRUCTURES에 폴리곤 2개 추가만
   - JuPedSim 물리 엔진이 자연스럽게 혼잡 재현
   - DO NOT REVERT 13개 항목과 충돌 없음
4. **문헌 근거**: Liao 2023 (Physica A), Helbing·Buzna 2005 (Transportation Science) — funnel 각도 최적 범위와 일치

### 구현 파라미터

본 연구 성수역 실제 적용 시 각 에스컬레이터 앞에:
- 진입부 폭: 1.0m (에스컬레이터 1인용 폭)
- 깔때기 시작 거리: 약 **5m** (멀리서부터 완만한 안내)
- 깔때기 각도: **15°** (좌우 대칭)
- 벽 두께: 0.1m (시각화용)

실제 좌표 (성수역 upper 출구 예시, 현재 x=27~28, y=24):
```python
upper_guide_left = Polygon([
    (22.0, 24.0), (27.0, 24.0),
    (27.0, 22.66),  # y = 24 - 1m (개구) - tan(15°) × 5m = 22.66
    (22.0, 22.66)
])
upper_guide_right = Polygon([
    (22.0, 24.0), (22.0, 25.34),  # 벽 바깥쪽 (외곽 포함)
    (27.0, 25.34), (27.0, 24.0)
])
```
(실제 적용 시 대합실 천장·구조물과의 기하 정합성 재검토 필요)

## 대안 1: 전략 C_range_×0.5 (반발 범위 축소)

### 선정 근거

- 진동 지표 **78% 감소**(208 → 44.8 deg/s, AVM 수준)
- 처리량 약 10% 감소 (34 → 31) — 수용 가능
- 단일 파라미터 변경으로 구현 비용 가장 낮음

### 구현

```python
# scenario_setup.py 또는 run_west_simulation_cfsm.py
CFSM_RANGE_NEIGHBOR = 0.05  # 기존 0.1 → 0.05 (×0.5)
```

### 주의

- **파라미터 변경이 다른 구간(게이트 앞, 자유보행)에도 영향**
- 가능하면 에스컬레이터 근처에서만 국소 적용해야 함. 그러나 JuPedSim 1.3.x는 에이전트별 파라미터 지원을 부분적으로만 제공 → **전역 변경 시 타 구간 V&V 재수행 필요**
- 권고 A 채택 시 부수 수정 불필요하므로, C는 **A로 충분하지 않을 때의 폴백**

## 검토 결과 기각한 전략

| 전략 | 기각 사유 |
|------|----------|
| B_speed (국소 속도 감소) | JuPedSim Agent API 한계로 구현 실패. 재구현 경로 필요 시 후속 과제로 검토 |
| D_AVM | 진동은 최소지만 **처리량 32% 감소** — 하류 병목을 "과도하게" 재현. RQ3(임계점 탐지)의 연속적 지표 신뢰성 저하 우려 |
| C_strength_×1.5 / ×2.0 | 진동 **악화** (역효과) |
| C_range_×2.0 | 진동 **크게 악화** |
| E_dt_0.025 | dt 축소로 진동 오히려 증가 + 계산시간 2배 — **구조적 문제(등방성)는 dt로 해결 안 됨** |
| F_funnel+dt | A 단독 대비 진동 10% 악화. 조합이 A 효과를 상쇄 |

## 후속 검증 권고 (본 작업 범위 밖)

1. **조합 F' = A_funnel_5m_15° + C_range_×0.5** 실험
   - 두 전략이 독립적 메커니즘이므로 시너지 가능
   - 진동 AVM 수준 + 처리량 baseline 수준 기대
2. **B 전략 재구현**: spawn 시점 v0를 영역 기반으로 초기화
3. **AVM 파라미터 튜닝**: anticipation_time 조정으로 단방향 병목용 캘리브레이션
4. **실제 성수역 시뮬 적용 시 회귀 테스트**: 13개 DO NOT REVERT 항목 유지 여부
5. **현장조사 데이터 대조**: 우이신설선 실측 영상에서 에스컬레이터 진입 행태 관측값과 시뮬 결과 비교

## 현 시점 결론

- **에스컬레이터 병목 재현 자체는 가능**. 베이스라인에서 유입률 4.0 ped/s 이상 시 진동 뚜렷
- **진동 악영향을 상쇄하는 가장 비용 효율적 방법 = Funnel geometry (A_funnel_5m_15°)**
- 본 권고는 **중간발표(2026-04-20) 이후 성수역 시뮬 확장 시** 적용
- 본 작업은 권고안 수립까지이며, 성수역 코드 (`simulation/`)에는 적용하지 않음
