## 1. 연구 배경

- 서울교통공사는 1~8호선 600개 통로에 태그리스 개찰구를 **단계적
  도입** 추진 중 (2023년 우이신설선 세계 최초 상용화).
- 단계적 도입 특성상 태그/태그리스 이용자가 혼재하는 **과도기**가
  장기간 지속 예상.
- 현행 운영: 역당 1~2대의 게이트가 **겸용**으로 지정, 나머지는 태그
  전용.
- 문제: 겸용 게이트에서 선행 태그 이용자의 정지·탭 동작이 후행
  태그리스 이용자의 **무정차 통과 이점을 상쇄**.

### 1-1. 기존 v1과의 차별 포인트

| 항목 | v1 (기존) | v2 (재정립) |
|------|-----------|-------------|
| 연구 초점 | 게이트 구간 통과시간 최적화 | **역사 전체** 통행비용 최적화 |
| 시뮬 공간 | 계단-대합실-게이트 | 계단-대합실-게이트-승강장/출구 |
| 평가 | 대기시간, 큐 길이 중심 | **밀도·LOS 가중 통합 비용** |
| 핵심 가설 | "전용 분리가 대기시간을 줄인다" | "전용 분리의 이점이 하류 혼잡으로 **상쇄**되는 임계점이 존재할 수 있다" |
| RQ 수 | 단순 K*(p) 도출 | RQ1(절감폭) + RQ2(단조/비단조) + RQ3(임계점 p*) |

## 2. 연구 목적

**주 RQ**: 성수역에서 태그리스 이용률 p%일 때, 역사 전체 통행비용을
최소화하는 게이트 배합은 무엇인가?

**하위 RQ**:
- **RQ1**: 현행 겸용 대비 분리 운영의 시스템 전체 통행비용 절감폭은?
- **RQ2**: 최적 배합은 이용률에 대해 단조인가, 임계점이 있는가?
- **RQ3** (본 연구의 핵심): 게이트 처리속도 향상이 하류 구간 밀도
  증가로 이어지는 임계점 p*이 존재하는가?

상세 기술은 `docs/연구설계_v2.md` §2 참조.

## 3. 시뮬레이션 공간 범위

```
[승강장] (간략: 도어 기반 생성 포인트)
    ↓ 열차 도착 이벤트
[계단 (upper/lower)]
    ↓
[대합실 (유료구역)]
    ↓ LRP 게이트 선택
[게이트 7대 (배합별 서비스 모델)]
    ↓
[대합실 (무료구역)]
    ↓
[출구 병목 (에스컬레이터 upper/lower)]
    ↓
[최종 출구]
```

- 승강장: **추가** (간략) — 열차 도착 이벤트 기반 생성
- 출구: **재설계** — 에스컬레이터 처리율 제약(1.1~1.3 ped/s)

상세 기술 및 DO NOT REVERT 충돌 분석은 `docs/시뮬_확장성_검토.md` 참조.

## 4. 연구 방법

### 4-1. 시뮬레이션 프레임워크

| 항목 | 내용 |
|------|------|
| 도구 | JuPedSim (Python, 오픈소스) |
| 보행 모델 | CFSM V2 (Tordeux et al. 2016) |
| 게이트 선택 | MNL (Gao et al. 2019 LRP 기반) |
| 후처리 분석 | PedPy (구간별 밀도/속도/흐름) |
| 대상역 | 성수역 2호선 서쪽 대합실 (50m × 25m, 게이트 7개) |
| V&V | NIST TN 1822 / ISO 20414 / RiMEA v4.1.1 |

### 4-2. 게이트 배합 (총 7대 고정)

| 배합 ID | 태그 전용 | 태그리스 전용 | 겸용 (버퍼) | 의도 |
|---------|-----------|--------------|-------------|------|
| B0 (현행) | 5~6 | 0 | 1~2 | 겸용 버퍼만 (혼재) |
| B1 (전체 겸용) | 0 | 0 | 7 | 관용 최대 |
| B2 (순수 분리) | 5 | 2 | 0 | 분리, 버퍼 없음 |
| B3 (분리+버퍼 소) | 4 | 2 | 1 | 분리 + 혼잡 흡수 1 |
| B4 (분리+버퍼 대) | 3 | 2 | 2 | 분리 + 버퍼 2 |

### 4-3. 이용자 유형별 Choice Set 차별화

| 게이트 유형 | 태그 이용자 | 태그리스 이용자 |
|------------|:---:|:---:|
| 태그 전용 | **O** | X |
| 태그리스 전용 | X | **O** |
| 겸용 (버퍼) | **O** | **O** |

- 겸용 게이트의 이용자 혼합 비율은 **외생적 가정 없이 MNL이
  내생적으로 결정**.
- Choice Set 차별화는 MNL 선택 효용함수의 가용 대안 집합에 반영.

### 4-4. 이용자 유형 및 서비스 시간

| 유형 | 통과 방식 | 서비스 시간 |
|------|-----------|-------------|
| 태그 | 정지→태핑→통과 | LogNormal, 평균 2.0s, 범위 [0.8, 3.7]s (Gao 2019) |
| 태그리스 | 무정차 연속 통과 | 1.2s 고정 (우이신설선 실측 후 재확정) |

### 4-5. 실험 설계

| 요인 | 수준 | 수 |
|------|------|-----|
| 게이트 배합 | B0~B4 | 5 |
| 태그리스 이용률 p | 10/20/40/60/80% | 5 |
| 시간대 | 첨두 / 비첨두 | 2 |
| **합계** | | **50 시나리오** |

- 각 시나리오 반복: 10회 (랜덤시드 0~9)
- 총 실행: **500회**
- SIM_TIME: 600s (열차 3편), 첫 180s 워밍업 제외

### 4-6. 통행비용 함수 ⭐

본 연구의 핵심 방법론. 세 후보 비교:

| 후보 | 수식 개요 | 문헌 근거 |
|------|-----------|-----------|
| (a) 가중합 | TC = α·T + β·∫ρ·dt | Wardman 2004, Tirachini 2013 |
| (b) LOS 페널티 | TC = T + Σ w_k·t_k | Fruin 1971, TCQSM 2013 |
| (c) Fruin 비선형 | TC = ∫ m(ρ)·dt | Fruin 1971, Weidmann 1993 |

**권고**: 주 지표 (c), 보조 (b), 민감도 (a). 상세: `docs/연구설계_v2.md` §4.

## 5. 평가 및 분석

### 5-1. 구간별 기본 지표

| 구간 | 지표 |
|------|------|
| 승강장 | 하차 완료 시간, 도어 분산 |
| 계단 | 시간당 흐름, 체류시간 |
| 대합실(유료) | 평균·최대 밀도, LOS 분포 |
| 게이트 | 처리량, 대기시간, 큐 길이, 균형도(MD) |
| 대합실(무료) | 밀도, 체류시간 |
| 출구 병목 | 진입 대기시간, 출구 앞 밀도 |
| 전체 | 총 통과시간, 완료율 |

### 5-2. 통합 지표 — 총 통행비용

- 4-6절 후보 (a)(b)(c) 모두 산출
- 시나리오별 평균 + 95% 신뢰구간
- 배합 × p 조합 Pareto frontier

### 5-3. 임계점 p* 탐지

1. 게이트 TC(p) vs 하류 TC(p) 교차점 탐색
2. $dTC/dp$ 부호 변화 지점 탐지
3. 배합별 p* 비교 → "버퍼 완충 효과" 정량화

## 6. 기대 결과 및 시사점

### 6-1. 학술적 기여
1. **다구간 보행 시뮬에서 병목 전이 정량 검증** (축 B gap 채움)
2. **이질 서비스율 혼재 운영의 시스템 최적화** 방법론 정립
3. **MNL Choice Set 차별화 + 시스템 비용 통합** 목적함수 제시

### 6-2. 정책적 함의
1. **과도기 게이트 배합 가이드라인**: p별 최적 배합 표
2. **시간대 가변 운영 전략**: 첨두/비첨두 배합 전환
3. **하드웨어 추가 없이 운영 변경만으로** 달성 가능한 효과 제시

### 6-3. 현실 운영과의 연결
- 서울교통공사 태그리스 확대 시 **배합 결정 근거 자료**
- 향후 적응형 동적 전환(후속 연구)의 정적 baseline
